# Kapasitif Dokunmatik Kontrol (TTP223)

**Seviye:** Uzman  
**Amaç:** Kapasitif Dokunmatik Kontrol (TTP223) projesini uygulamak.

## Bileşenler
- Arduino Uno
- Breadboard
- Jumper kablolar
- İlgili sensör/modüller


## Bağlantılar (Özet)
- Sensör VCC -> 5V
- GND -> GND
- Sinyal/Giriş pinleri -> dijital/analog pinler

## Kod
`proje.ino` dosyasını yükleyin.

## Notlar
Bu proje için ayrıntılı şema ve nihai kod proje kapsamına göre eklenecektir.
