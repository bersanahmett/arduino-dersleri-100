# 2 Eksenli Joystick ile Servo Kontrol

**Seviye:** Orta  
**Amaç:** 2 Eksenli Joystick ile Servo Kontrol projesini uygulamak.

## Bileşenler
- Arduino Uno
- Breadboard
- Jumper kablolar
- İlgili sensör/modüller


## Bağlantılar (Özet)
- Sensör VCC -> 5V
- GND -> GND
- Sinyal/Giriş pinleri -> dijital/analog pinler

## Kod
`proje.ino` dosyasını yükleyin.

## Notlar
Bu proje için ayrıntılı şema ve nihai kod proje kapsamına göre eklenecektir.
