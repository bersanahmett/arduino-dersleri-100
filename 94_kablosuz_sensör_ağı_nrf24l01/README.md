# Kablosuz Sensör Ağı (nRF24L01)

**Seviye:** Uzman  
**Amaç:** Kablosuz Sensör Ağı (nRF24L01) projesini uygulamak.

## Bileşenler
- Arduino Uno
- Breadboard
- Jumper kablolar
- İlgili sensör/modüller



## Bağlantılar (Özet)
- Sensör VCC -> 5V
- GND -> GND
- Sinyal/Giriş pinleri -> dijital/analog pinler

## Kod
`proje.ino` dosyasını yükleyin.

## Notlar
Bu proje için ayrıntılı şema ve nihai kod proje kapsamına göre eklenecektir.
