
# Ultrasonik ile Mesafe Ölçümü

**Seviye:** Orta  
**Amaç:** HC-SR04 ile cm cinsinden mesafe ölçmek.

## Bileşenler
- Arduino Uno
- HC-SR04 ultrasonik sensör
- Kablolar



## Bağlantılar (Özet)
- HC-SR04 VCC -> 5V
- HC-SR04 GND -> GND
- Trig -> D9
- Echo -> D10

## Kod
`proje.ino` dosyasını yükleyin.

## Notlar
Echo pini 5V döner; bazı kartlarda seviye dönüştürücü gerekebilir.
