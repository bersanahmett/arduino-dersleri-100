# Akıllı Sera Otomasyonu

**Seviye:** İleri  
**Amaç:** Akıllı Sera Otomasyonu projesini uygulamak.

## Bileşenler
- Arduino Uno
- Breadboard
- Jumper kablolar
- İlgili sensör/modüller



## Bağlantılar (Özet)
- Sensör VCC -> 5V
- GND -> GND
- Sinyal/Giriş pinleri -> dijital/analog pinler

## Kod
`proje.ino` dosyasını yükleyin.

## Notlar
Bu proje için ayrıntılı şema ve nihai kod proje kapsamına göre eklenecektir.
